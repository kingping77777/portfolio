# Daksh Sengar — Portfolio

Personal portfolio website built with SvelteKit and Svelte 5.

A presentation SPA showcasing my work in AI & Machine Learning, Data Science, Web Development and Research.

Built with a focus on performance and creative motion design. Features include font subsetting, custom scroll animations, minimal external dependencies and i18n support (English/Hindi).

## Tech Stack

- **Framework**: SvelteKit + Svelte 5
- **Styling**: SCSS
- **Animations**: Custom scroll-based animations
- **i18n**: Paraglide.js (English + Hindi)
- **Deployment**: Vercel

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
